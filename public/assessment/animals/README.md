# Assessment Animal Images

Place optional animal images for the sleep question in this folder.

Required filenames:

- `sleep-bear.jpg`
- `sleep-cat.jpg`
- `sleep-owl.jpg`
- `sleep-jellyfish.jpg`

Recommended style: warm, calm, editorial wellness photography or soft illustration, 16:9 crop, at least 1200 x 675 px.

If a file is missing, the assessment skips that image without showing an error.
